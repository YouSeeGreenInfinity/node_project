Как запустить проект:
1. Заполнить env файл на сервере (user-service-backend)
2. Если хотите посмотреть и фронтенд и бекенд, то они запускаются одной командой, в общем package.json написаны скрипты
(В сидах в бд есть почта/пароль админа в сидах: admin@example.com / Admin123. Можно протестировать личный кабинет админа)
3. Если нужен только бекенд, то удалите папку client. Client  сделал чисто для себя, чтобы протренироваться.
